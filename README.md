# ENTROPY: Руководство Пользователя

Добро пожаловать в систему **ENTROPY** — интерактивный реестр персонажей для D&D в уникальной двойной стилистике. Это приложение позволяет создавать, редактировать и хранить анкеты персонажей, выбирая между двумя фракциями: **Империя (Ordo Continuum)** и **Сопротивление**.

## 1. Выбор Стороны

При входе на сайт вы попадаете на экран выбора стороны.
*   **Империя (Правая сторона):** Строгий золотой стиль, порядок, гексагональные сетки.
*   **Сопротивление (Левая сторона):** Хакерский зеленый стиль, глитчи, эффект "шума" и фонарика.

**Как войти:**
Просто наведите курсор на желаемую половину экрана и кликните. Экран трансформируется, открывая доступ к соответствующему реестру.

---

## 2. Реестр (Список Персонажей)

Здесь хранятся все ваши досье.

*   **Создание:** Нажмите на большую карту с плюсом `[+]` или надписью `НОВЫЙ ПРОТОКОЛ` / `INIT NEW PROTOCOL`. Введите имя персонажа в появившемся окне.
*   **Открыть:** Кликните по карточке персонажа, чтобы открыть его полное досье.
*   **Удалить:**
    *   *Империя:* Наведите на карточку, в углу появится крестик `×`.
    *   *Сопротивление:* Нажмите кнопку `[DEL]` в углу карточки.
    *   *Внимание:* Удаление требует текстового подтверждения.

---

## 3. Досье Персонажа

Анкета разделена на 7 основных вкладок. Все изменения сохраняются автоматически.

### Управление и Интерфейс
*   **Редактирование:** Большинство полей редактируются по клику. Для списков (умения, инвентарь) нажмите на название элемента, чтобы открыть подробный редактор.
*   **Счетчики:** Числовые поля имеют стилизованные стрелочки (вверх/вниз) для изменения значений.
*   **Перетаскивание (Drag & Drop):** Элементы в списках (оружие, заклинания, черты) имеют иконку (точки или полоски) справа. Зажмите её и перетяните элемент, чтобы изменить порядок.
*   **Блокировка вкладок:** Рядом с названием каждой вкладки есть маленький квадратик/тумблер. Нажав на него, вы скроете содержимое вкладки (появится надпись `ACCESS DENIED`). Это удобно для стримов или сокрытия спойлеров.

### Вкладки

#### I. Главная (Identity)
Основная информация: Имя, Раса, Класс, Уровень.
*   **Фото:** Нажмите кнопку редактирования (или иконку обновления) на рамке фото, чтобы вставить ссылку на изображение.
*   **Бонус мастерства:** Рассчитывается автоматически от Уровня.

#### II. Характеристики (Stats)
*   **Основные статы:** Сила, Ловкость и т.д. Модификаторы считаются автоматически.
*   **Жизнедеятельность:**
    *   *Хиты:* Введите текущее и максимальное значение.
    *   *Временные хиты:* Введите значение в оранжевое поле, оно отобразится поверх шкалы здоровья.
    *   *Щиты:* Дополнительная синяя шкала (аналог силового поля).
*   **Спасброски:** Отметьте галочками те, которыми владеете.

#### III. Возможности (Features)
Разделена на подкатегории:
*   **Навыки:** Таблица всех навыков.
    *   Первая галочка = Владение (Proficiency).
    *   Вторая галочка = Компетентность (Expertise).
    *   Поле ввода = Дополнительный бонус.
*   **Умения / Черты / Особенности:** Списки для добавления классовых способностей. Можно добавлять, удалять и менять местами.
*   **Владения:** Языки, Инструменты, Броня.

#### IV. Снаряжение (Gear)
*   **Оружие:** Таблица атак. Укажите название, урон, тип и свойства.
*   **Инвентарь:** Список предметов.
*   **Финансы:** Поля для валюты. Снизу автоматически считается общая стоимость в золотом эквиваленте.

#### V. Данные (Psych/Data)
Ролевая информация: Возраст, Рост, Вес, Идеалы, Привязанности, Слабости и текстовое поле для Квенты/Анализа.

#### VI. Псионика (Psionics)
Автоматизированный калькулятор магии/псионики.
1.  **Параметры:**
    *   Выберите **Базовую характеристику** (Интеллект, Мудрость, Харизма). Это определит Тип псионики (Учёная или Интуитивная).
    *   Выберите **Тип кастера** (Полный, Половинный, Третичный).
    *   Система автоматически рассчитает ваш **Класс Псионики**, **Сложность (DC)** спасброска, количество известных **Заговоров** и **Заклинаний**.
    *   Шкала **Пси-очков** показывает ваш текущий запас маны/энергии.
2.  **Заклинания:**
    *   Добавляйте заклинания в список.
    *   Если стоимость (Cost) = 0, заклинание попадет в раздел "ЗАГОВОРЫ".
    *   Если стоимость > 0, оно попадет в раздел "ЗАКЛИНАНИЯ".

#### VII. Универсальное (Universalis)
*   **Кастомный спасбросок:** Если вашему персонажу нужно отслеживать уникальную сложность (например, для гаджетов), настройте базу и атрибут здесь.
*   **Счетчики:** Создавайте любые счетчики (стрелы, заряды способностей, ячейки).
*   **Таблица:** Простой список для заметок.

## 4. Оффлайн режим

Приложение использует базу данных для синхронизации. Однако, если у вас пропадет интернет или сервер будет недоступен:
1.  Внизу экрана появится надпись `LOCAL (OFFLINE)` или `OFFLINE (CACHE)`.
2.  Приложение автоматически переключится на **Локальное хранилище** (память браузера).
3.  Вы сможете продолжать создавать и редактировать персонажей. Эти данные сохранятся только на *этом* устройстве и в *этом* браузере.
